University of Windsor Data API - Guide

How to talk to the API, and what endpoints exist :)



Notes:

The "parameter" referred to below is given by the title "q" in a GET request
Example: /api/courses?q=COMP-1000 returns information on COMP 1000
Leave the parameter empty to retrieve all data


Datasets and API Endpoints:

# Dataset API Endpoint Parameter Description
1 Courses /api/courses Course code Contains course code, title, and description
2 Staff /api/staff Staff name Contains name, email, phone, title, department, and location
3 Buildings /api/buildings Building name Contains building name and location (latitude and longitude)
4 Holidays /api/holidays Date (in format YYYY-MM-DD) Contains date and what the holiday/date is
5 Emergency Services /api/emergency_services Service name (ex. Campus Police) Contains phone, extension, service name, and website
